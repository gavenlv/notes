# Day 12: 安全与权限管理

在企业级 Ansible 应用中，安全和权限管理是至关重要的。本日学习将深入探讨 Ansible Vault、权限控制和安全加固技术。

## 📚 学习内容

- Ansible Vault 的使用和管理
- 敏感数据加密和解密
- 权限模型和访问控制
- 安全加固最佳实践
- 合规性检查和审计

## 📁 目录结构

```
day12-security-permissions/
├── examples/           # 安全配置示例
├── playbooks/          # 安全相关的 Playbook
├── scripts/            # 安全管理脚本
└── security.md         # 详细学习文档
```

## 🎯 学习目标

完成本日学习后，您将能够：

1. 使用 Ansible Vault 加密和管理敏感数据
2. 实施细粒度的权限控制策略
3. 配置安全加固措施保护基础设施
4. 执行合规性检查和安全审计
5. 建立企业级安全管理体系

## 🚀 快速开始

1. 查看 `security.md` 了解详细的安全管理技术
2. 运行 `examples/` 中的示例了解基本用法
3. 实践 `playbooks/` 中的完整安全配置方案
4. 使用 `scripts/` 中的工具进行安全管理
