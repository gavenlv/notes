## 如何使用本教程

1. **按顺序学习**：建议从第1章开始，按顺序学习，逐步建立完整的测试知识体系
2. **理论结合实践**：每章都包含理论文档和示例代码，建议先阅读理论，再运行代码示例
3. **动手实践**：尝试修改示例代码，完成每章中的实验，加深理解
4. **扩展学习**：参考学习资源汇总，选择适合自己的进阶学习材料

## 运行示例代码

所有示例代码都位于`code`目录下，每个章节对应一个Python文件：

```bash
# 运行第1章示例代码
python code/chapter1_examples.py

# 运行第2章示例代码
python code/chapter2_examples.py

# ...以此类推
```
