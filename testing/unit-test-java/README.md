# Java单元测试从入门到专家

## 项目介绍

本项目是一个完整的Java单元测试学习教程，从基础概念到高级实践，帮助开发者掌握Java单元测试技能，提高代码质量和开发效率。

## 教程结构

本教程包含8个核心章节，每个章节都有详细的概念解析、完整的代码示例和最佳实践指导。

### 章节内容

1. **Java单元测试简介与环境搭建** - 单元测试基础概念、环境配置、第一个测试
2. **JUnit基础概念与核心注解** - JUnit框架、核心注解使用、测试结构
3. **JUnit断言与验证机制** - 断言方法、异常测试、条件测试
4. **JUnit测试生命周期与钩子方法** - 测试生命周期、设置与清理、测试顺序
5. **Mockito框架与模拟对象** - Mock对象、存根、验证交互
6. **参数化测试与测试套件** - 参数化测试、测试套件、重复测试
7. **测试驱动开发(TDD)实践** - TDD流程、重构技巧、实践案例
8. **Java单元测试最佳实践与代码质量** - 测试策略、代码覆盖率、持续集成

### 代码示例

每个章节都包含完整的可运行代码示例，位于`code/chapterX/`目录下，可以直接导入IDE运行。

## 学习路径

### 初学者路径 (1-2周)
- 阅读第1-3章，理解基础概念
- 实践基础测试编写和断言使用
- 完成简单项目的单元测试

### 进阶路径 (2-4周)
- 学习第4-6章的高级功能
- 实践Mockito和参数化测试
- 掌握测试生命周期管理

### 专家路径 (4-8周)
- 深入第7-8章的TDD实践
- 学习测试策略和最佳实践
- 参与实际项目的单元测试改进

## 技术栈

- **Java**: 8+ (推荐Java 11或更高版本)
- **JUnit**: 5.x (最新版本)
- **Mockito**: 4.x (最新版本)
- **Maven**: 构建工具
- **IntelliJ IDEA**: 开发IDE (推荐)

## 使用说明

1. **环境准备**: 确保已安装JDK 8+和Maven
2. **代码导入**: 将各章节代码导入IDE
3. **运行测试**: 直接运行JUnit测试类
4. **学习实践**: 按章节顺序学习并实践
