# 第9章 — 高级主题：Property-based 与 Mutation Testing

目标：
- 了解更高级的测试技术：property-based testing（基于性质的测试）和 mutation testing（变异测试）。

Property-based testing
- 使用 `hypothesis` 来生成大量随机输入并检查性质（不只是具体示例）。
- 适合检查广泛不变量，例如“对称性”、“可逆操作” 等。

Mutation testing
- 使用变异测试工具（如 `mutmut`、`cosmic-ray`）来验证测试套件是否能捕获小的代码变异。
- 变异测试能帮助发现测试盲点。

注意
- 这些工具需要额外依赖并通常用于成熟项目，初学阶段先掌握 unit tests 的设计与隔离更为重要。

练习
- 阅读 `hypothesis` 文档并尝试为 `is_palindrome` 写一个 property-based 测试。
