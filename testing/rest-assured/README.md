# REST Assured 从入门到专家

本教程提供完整的REST Assured学习路径，从基础概念到高级应用，帮助您掌握RESTful API测试的核心技能。

## 教程结构

- **第1章：REST Assured简介与环境搭建** - 介绍REST Assured的基本概念、特点和项目搭建
- **第2章：REST Assured基础语法与核心概念** - 深入讲解请求构建、响应验证和核心API
- **第3章：请求构建与参数处理** - 详细介绍请求头、查询参数、路径参数和请求体设置
- **第4章：响应验证与断言** - 全面讲解状态码、响应头、响应体验证和自定义断言
- **第5章：JSON与XML数据处理** - 深入探索JSONPath、XML路径和复杂数据结构处理
- **第6章：身份验证与安全测试** - 介绍各种认证机制和安全测试最佳实践
- **第7章：测试框架设计与组织** - 构建可维护、可扩展的测试框架
- **第8章：高级特性与最佳实践** - 探讨高级功能、性能优化和企业级应用

## 代码示例

所有代码示例位于 `code` 目录下，按章节和功能分类：

- **basic-examples** - 基础语法和核心概念示例
- **request-response** - 请求构建和响应验证示例
- **data-handling** - JSON和XML数据处理示例
- **authentication** - 身份验证和安全测试示例
- **framework** - 测试框架设计示例
- **advanced** - 高级特性和最佳实践示例

## 快速开始

1. 阅读第1章了解基本概念和环境搭建
2. 查看快速开始指南 (`QUICK_START.md`)
3. 运行基础示例验证环境
4. 按章节顺序深入学习

## 先决条件

- Java 8或更高版本
- Maven或Gradle构建工具
- 基本的HTTP和REST API知识
- Java编程基础

## 学习建议

1. **理论与实践结合** - 每个概念都配有可运行的代码示例
2. **由浅入深** - 按章节顺序学习，循序渐进
3. **实践为主** - 运行并修改代码示例，加深理解
4. **项目驱动** - 尝试构建自己的测试项目
5. **持续练习** - 在实际项目中应用所学知识
