## 学习目标

1. 理解k6的核心概念和架构
2. 掌握k6脚本的编写方法
3. 学会使用k6进行各种类型的性能测试
4. 掌握k6在生产环境中的最佳实践
5. 能够独立设计和实施专业的性能测试方案

## 前置要求

- 基本的编程概念（变量、函数、控制流）
- 基础的HTTP协议知识
- 命令行操作基础

## 工具与环境

- k6 v0.50.0+
- Node.js（可选，用于JavaScript脚本）
- Git（用于代码管理）
- 终端/命令行工具
