## 使用说明

### 运行单个章节的代码示例

1. 进入章节目录：
   ```bash
   cd testing/selenium/code/chapter1
   ```

2. 安装依赖：
   ```bash
   mvn clean install
   ```

3. 运行测试：
   ```bash
   mvn test
   ```

### 运行特定测试类

1. 进入章节目录

2. 运行特定测试类：
   ```bash
   mvn test -Dtest=FirstSeleniumTest
   ```

### 运行TestNG套件

1. 进入章节目录

2. 运行TestNG套件：
   ```bash
   mvn test -DsuiteXmlFile=testng.xml
   ```

## 项目依赖

每个章节的代码示例都有独立的Maven配置，主要依赖包括：

- Selenium WebDriver
- TestNG
- WebDriverManager
- AssertJ（流畅断言）
- Log4j2/SLF4J（日志框架）
- Jackson（JSON处理）
- Apache POI（Excel处理）
- ExtentReports/Allure（报告生成）

## 注意事项

1. 每个章节的代码示例都是独立的，可以单独运行
2. 部分章节需要额外的配置（如数据库连接、云服务账户等）
3. 建议按照章节顺序学习，逐步深入
4. 在运行代码前，请确保已安装必要的依赖和环境
5. 对于涉及外部服务的章节，可能需要相应的账户或服务访问权限
