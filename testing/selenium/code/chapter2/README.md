# 第2章代码示例 - Selenium基础概念与WebDriver API

本章包含以下代码示例：

1. **WebDriver API基础操作**
   - `WebDriverApiTest.java` - WebDriver基础操作演示
   - `NavigationTest.java` - 页面导航操作示例

2. **元素定位与操作**
   - `ElementLocationTest.java` - 各种元素定位策略示例
   - `WebElementTest.java` - WebElement操作示例

3. **JavaScript交互**
   - `JavaScriptExecutorTest.java` - JavaScript执行示例

4. **高级功能**
   - `ScreenshotTest.java` - 截图功能示例
   - `BrowserOptionsTest.java` - 浏览器选项和Cookie管理示例

5. **工具类**
   - `ScreenshotUtil.java` - 截图工具类
   - `BrowserUtils.java` - 浏览器操作工具类

## 运行指南

1. 确保已完成第1章环境搭建
2. 运行各类测试了解WebDriver API的使用
3. 重点关注元素定位、JavaScript交互和截图功能

## 注意事项

- 部分测试使用外部网站，确保网络连接正常
- 截图功能会在screenshots目录下生成图片文件
- 部分测试演示了异常处理和错误恢复机制
