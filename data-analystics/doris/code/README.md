## 🚀 快速开始

每个章节的代码示例都包含：

1. **SQL脚本** - 可直接在Doris中执行的SQL语句
2. **配置文件** - 相关配置示例
3. **Shell脚本** - 自动化脚本示例
4. **README.md** - 代码说明和使用指南

## 📖 使用说明

1. 确保已安装并启动Doris集群
2. 根据章节说明执行相应的代码示例
3. 参考每个章节的README.md了解详细使用方法

## 🔧 环境要求

- Apache Doris 1.2+ 版本
- MySQL客户端或其他支持MySQL协议的客户端工具
- Linux/macOS/Windows 操作系统
