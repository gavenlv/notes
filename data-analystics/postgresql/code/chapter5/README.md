# 第5章代码示例

本目录包含PostgreSQL第5章"高级查询技术"的所有代码示例：

1. `join_examples.sql` - 连接查询示例
2. `subquery_examples.sql` - 子查询示例
3. `set_operations.sql` - 集合操作示例
4. `window_functions.sql` - 窗口函数示例
5. `comprehensive_example.sql` - 综合示例
6. `python_examples.py` - Python中使用高级查询技术的示例
7. `requirements.txt` - Python依赖包

## 运行说明

1. 确保PostgreSQL服务器正在运行
2. 创建一个新的数据库用于测试
3. 按顺序运行SQL文件以创建表结构和示例数据
4. 执行各个示例文件中的查询语句
5. 运行Python示例前需要安装依赖：`pip install -r requirements.txt`
