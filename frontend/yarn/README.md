## 学习路径

### 初学者（第1-3章）
- Yarn基础概念与安装
- 基本命令使用
- 项目依赖管理

### 进阶开发者（第4-6章）
- 脚本自动化
- 高级配置技巧
- 插件系统

### 高级开发者（第7-10章）
- Yarn Plug'n'Play深入
- Yarn Berry新特性
- 企业级应用与性能优化

## 环境要求

- Node.js 14.0+ 
- Yarn 1.22+（Yarn Berry部分需要Yarn 2.0+）
- 基本的命令行操作知识
