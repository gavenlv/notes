# 第一章代码示例说明

本章的代码示例展示了如何创建和运行一个基本的Vue.js应用程序。

## 目录结构

```
chapter1-demo/
├── public/
│   ├── index.html
│   └── favicon.ico
├── src/
│   ├── assets/
│   │   └── logo.png
│   ├── components/
│   │   └── HelloWorld.vue
│   ├── App.vue
│   └── main.js
├── package.json
└── README.md
```

## 示例说明

### 1. 基础Vue应用
展示了最基本的Vue.js应用结构，包含：
- Vue实例的创建和挂载
- 模板语法的基本使用
- 数据绑定和事件处理

### 2. 组件化示例
演示了Vue组件的基本用法：
- 组件的定义和使用
- Props传递数据
- 组件间的通信

## 运行说明

1. 确保已安装Node.js和npm
2. 在项目根目录下运行以下命令安装依赖：
   ```
   npm install
   ```
3. 启动开发服务器：
   ```
   npm run serve
   ```
4. 在浏览器中访问 http://localhost:8080 查看效果
