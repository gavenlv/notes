文本对齐示例

.text-left (默认)

这是一段左对齐的文本。Tailwind CSS默认的文本对齐方式是左对齐，这在大多数情况下都是合适的选择。

.text-center

这是一段居中对齐的文本。居中对齐常用于标题、重要信息或需要强调的内容，可以使内容在视觉上更加突出。

.text-right

这是一段右对齐的文本。右对齐通常用于日期、价格或其他需要在右侧对齐的信息，在表格中也经常使用。

行高和字间距示例

行高示例

.leading-tight这是一段使用紧密行高的文本，适合标题或需要紧凑布局的地方。

.leading-normal这是一段使用正常行高的文本，是最常用的行高设置，适合大多数正文内容。

.leading-relaxed这是一段使用宽松行高的文本，提供更好的可读性，适合长段落或需要提高阅读舒适度的内容。

字间距示例

.tracking-tight紧凑字间距。

.tracking-normal正常字间距。

.tracking-wide宽松字间距。

.uppercase .tracking-wider大写字母宽间距。
