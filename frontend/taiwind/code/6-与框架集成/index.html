集成指南

React 集成步骤

  1. 使用 Vite 创建 React 项目
  2. 安装 Tailwind CSS 及其依赖
  3. 初始化 Tailwind 配置
  4. 配置内容路径
  5. 导入 Tailwind 指令
  6. 在组件中使用 Tailwind 类

Vue 集成步骤

  1. 使用 Vite 创建 Vue 项目
  2. 安装 Tailwind CSS 及其依赖
  3. 初始化 Tailwind 配置
  4. 配置内容路径
  5. 导入 Tailwind 指令
  6. 在组件中使用 Tailwind 类

Angular 集成步骤

  1. 使用 Angular CLI 创建项目
  2. 安装 Tailwind CSS 及其依赖
  3. 初始化 Tailwind 配置
  4. 配置 PostCSS
  5. 导入 Tailwind 指令
  6. 在组件中使用 Tailwind 类

最佳实践

组件设计

  • 保持组件结构清晰，样式与逻辑分离
  • 使用语义化 HTML 标签
  • 遵循单一职责原则
  • 提取可复用的组件类

样式管理

  • 使用 @layer 指令组织样式
  • 自定义主题保持设计一致性
  • 避免内联样式，优先使用 Tailwind 类
  • 启用 JIT 模式提高性能
