配置优化与智能提示

优化的配置文件

// tailwind.config.js
module.exports = {
  mode: 'jit',
  content: [
    './src/**/*.{html,js,jsx,ts,tsx}',
  ],
  theme: {
    extend: {
      colors: {
        primary: '#3B82F6',
        secondary: '#10B981',
      },
      fontFamily: {
        sans: ['Inter', 'system-ui'],
      },
      fontSize: {
        'fluid-lg': 'clamp(1.25rem, 6vw, 1.75rem)',
      },
    },
  },
  plugins: [
    require('@tailwindcss/typography'),
    require('@tailwindcss/forms'),
  ],
}

性能优化技巧

JIT 编译

启用 JIT 编译模式，只生成实际使用的 CSS 类，显著减少 CSS 文件大小。

// tailwind.config.js
module.exports = {
  mode: 'jit',
  content: [
    './src/**/*.{html,js,jsx,ts,tsx}',
  ],
}

按需加载

结合代码分割功能，按需加载 Tailwind 样式，减少初始加载时间。

// React 中按需导入
import './styles/tailwind.css';

// 或者使用 CSS-in-JS
import { tw } from 'twind';

const Component = () => {
  return <div className={tw`bg-blue-500`}>Hello</div>;
};

图片优化

使用响应式图片和延迟加载技术，减少带宽消耗和加载时间。

<img src="small.jpg"
     srcset="small.jpg 640w, medium.jpg 1280w"
     sizes="(max-width: 640px) 100vw, 50vw"
     loading="lazy"
     alt="响应式图片">

可访问性最佳实践

语义化 HTML

使用适当的语义化 HTML 标签，提高屏幕阅读器的可访问性。

<!-- 语义化导航 -->
<nav aria-label="主导航">
  <ul>
    <li><a href="#" aria-current="page">首页</a></li>
    <li><a href="#">关于</a></li>
  </ul>
</nav>

<!-- 语义化卡片 -->
<article class="card">
  <header><h2>卡片标题</h2></header>
  <div>卡片内容</div>
  <footer>卡片页脚</footer>
</article>

颜色对比度与键盘导航

确保文本和背景之间有足够的对比度，并支持键盘导航。

<!-- 好的对比度 -->
<div class="bg-white text-gray-800 p-4">
  易于阅读的文本
</div>

<!-- 支持键盘导航 -->
<button 
  class="btn btn-primary"
  tabindex="0"
  aria-label="主要按钮"
>
  主要按钮
</button>
