# React Hooks 演示应用

这个目录包含了React核心概念中关于Hooks使用的示例代码。

## 如何运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

## 文件说明

- `src/App.jsx`：主应用组件，整合所有演示功能
- `src/hooks/`：自定义Hooks目录
- `src/components/`：示例组件目录
- `src/utils/`：工具函数

## 学习目标

通过这个示例，你将学习：
- useState Hook的使用和状态管理
- useEffect Hook的使用和副作用处理
- useContext Hook的使用和上下文共享
- useReducer Hook的使用和复杂状态管理
- 自定义Hook的创建和复用
- 表单处理和事件处理
