# 基础React应用示例

这个目录包含了React基础应用的示例代码。

## 如何运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

## 文件说明

- `index.html`：HTML入口文件
- `src/main.jsx`：React应用入口文件
- `src/App.jsx`：主应用组件
- `src/components/`：存放可复用组件
- `src/styles/`：存放CSS样式文件

## 学习目标

通过这个示例，你将学习：
- 如何创建基本的React组件
- JSX语法的使用
- 组件的渲染和挂载
- 基本的项目结构组织
