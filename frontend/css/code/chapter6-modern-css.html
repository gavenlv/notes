3. CSS 变量高级应用

CSS 变量（自定义属性）允许我们定义可重用的值，并在整个样式表中引用它们，使得主题切换和样式管理变得更加简单。

Theme Demo

这个示例展示了如何使用 CSS 变量来管理主题颜色和样式。

/* CSS 变量定义 */ :root { --primary-color: #3498db; --secondary-color: #2ecc71; --text-color: #333; --background-color: #f9f9f9; --card-bg-color: #ffffff; } /* 暗黑主题变量 */ :root.dark-theme { --primary-color: #3498db; --secondary-color: #2ecc71; --text-color: #ffffff; --background-color: #1a1a1a; --card-bg-color: #2d2d2d; } /* 使用变量 */ .theme-button { background-color: var(--primary-color); color: white; transition: background-color var(--transition-speed) ease; }

4. 容器查询 (Container Queries)

容器查询允许元素基于其包含块的尺寸而不是视口尺寸来改变其样式，为组件级响应式设计提供了更精确的控制。

容器查询示例

调整容器大小，内部元素会根据容器尺寸自动改变布局。

/* 容器查询 */ .container-query-demo { container-type: inline-size; } /* 小容器时垂直排列 */ @container (max-width: 500px) { .container-responsive { flex-direction: column; } } /* 大容器时水平排列 */ @container (min-width: 501px) { .container-responsive { display: flex; gap: 16px; } .container-responsive-item { flex: 1; } }

5. 逻辑属性与值

逻辑属性和值基于文档的写入模式、方向和文本方向，提供了更加灵活和国际化友好的布局方式。

逻辑属性示例

这个盒子使用了逻辑属性，如 margin-block、padding-inline 等，而不是物理属性。

逻辑属性让布局更加灵活，适应不同的文本方向和书写模式。

/* 逻辑属性示例 */ .logical-box { /* 使用逻辑属性替代物理属性 */ margin-block: 20px; /* 替代 margin-top, margin-bottom */ margin-inline: auto; /* 替代 margin-left, margin-right */ padding-block: 24px; /* 替代 padding-top, padding-bottom */ padding-inline: 32px; /* 替代 padding-left, padding-right */ border-block-start: 4px solid var(--primary-color); border-inline-end: 4px solid var(--secondary-color); max-inline-size: 500px; /* 替代 max-width */ }
