CSS高级选择器与特效示例演示

1. CSS高级选择器

属性选择器

重要提示：使用[data-type="important"]选择器
信息提示：使用[data-type="info"]选择器
警告信息：使用[class^="warning"]选择器
内容区块：使用[class$="-box"]选择器
/* 属性选择器示例 */ /* 精确属性值匹配 */ [data-type="important"] { background-color: #fff3cd; border-left: 4px solid #ffc107; } [data-type="info"] { background-color: #d1ecf1; border-left: 4px solid #17a2b8; } /* 属性值开头匹配 */ [class^="warning"] { background-color: #fff3cd; border-left: 4px solid #ffc107; } /* 属性值结尾匹配 */ [class$="-box"] { background-color: #e2e3e5; border-left: 4px solid #6c757d; } /* 属性值包含匹配 */ [class*="message"] { font-weight: bold; } /* 属性存在匹配 */ [title] { cursor: help; } /* 多属性选择器 */ input[type="text"][required] { border-color: #ff0000; }

伪类选择器进阶

/* 伪类选择器进阶示例 */ /* 表单状态伪类 */ input:focus { border-color: var(--primary-color); box-shadow: 0 0 0 3px rgba(52, 152, 219, 0.25); outline: none; } input:valid { border-color: var(--secondary-color); } input:invalid { border-color: var(--accent-color); } /* 否定伪类 */ :not(.special) { opacity: 0.7; } /* 任意匹配伪类 */ :any-link { color: var(--primary-color); } /* 焦点内伪类 */ :focus-within { background-color: rgba(52, 152, 219, 0.05); } /* 只读/可写伪类 */ input:read-only { background-color: #f8f9fa; } input:read-write { background-color: white; } /* 选中伪类 */ input:checked + label { font-weight: bold; } /* 目标伪类 */ :target { background-color: #fff3cd; padding: 10px; }

2. CSS变量 (自定义属性)

使用CSS变量定义的主题颜色
变量名: --primary-color, --secondary-color
/* CSS变量示例 */ /* 定义根级变量 */ :root { --primary-color: #3498db; --secondary-color: #2ecc71; --accent-color: #e74c3c; --text-color: #333; --background-color: #f5f5f5; --transition-speed: 0.3s; } /* 使用变量 */ .element { color: var(--text-color); background-color: var(--primary-color); transition: all var(--transition-speed) ease; } /* 局部变量 */ .section { --section-bg: #fff; background-color: var(--section-bg); } /* 变量计算 */ .element { width: calc(var(--base-size) * 2); padding: calc(var(--spacing) + 10px); } /* 变量回退值 */ .element { color: var(--non-existent-color, #333); } /* JavaScript操作变量 */ /* document.documentElement.style.setProperty('--primary-color', '#ff0000'); */
