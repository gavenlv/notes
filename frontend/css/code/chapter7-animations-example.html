复杂的多步骤动画

多步骤动画：移动 + 旋转 + 淡出

@keyframes multiStep { 0% { transform: translateX(-100%) rotate(0deg); opacity: 0; } 30% { transform: translateX(0) rotate(0deg); opacity: 1; } 60% { transform: translateX(0) rotate(360deg); opacity: 1; } 100% { transform: translateX(100%) rotate(360deg); opacity: 0; } }
