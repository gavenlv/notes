CommonJS vs ES Modules

// commonjs/math.js
const PI = 3.14159;

function add(a, b) {
  return a + b;
}

module.exports = {
  PI,
  add
}; // commonjs/main.js
const { PI, add } = require('./math.js');
console.log(add(5, 3)); // 8
// esmodules/math.js
export const PI = 3.14159;

export function add(a, b) {
  return a + b;
}
// esmodules/main.js
import { PI, add } from './math.js';
console.log(add(5, 3)); // 8
主要区别:

1. 加载方式:
- CommonJS: 同步加载，适用于服务器
- ES Modules: 异步加载，适用于浏览器

2. 导出语法:
- CommonJS: module.exports / exports
- ES Modules: export / export default

3. 导入语法:
- CommonJS: require()
- ES Modules: import

4. 值的引用:
- CommonJS: 导出的是值的拷贝
- ES Modules: 导出的是值的引用
