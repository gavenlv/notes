## 🚀 如何使用这些示例

### 1. 直接在浏览器中运行

每个示例都是一个独立的HTML文件，可以直接在现代浏览器中打开：

1. 找到对应的HTML文件（如`chapter1_javascript_basics.html`）
2. 双击文件或右键选择"在浏览器中打开"
3. 在浏览器中查看并交互

### 2. 使用本地服务器（推荐）

为了避免跨域问题和确保最佳体验，建议使用本地服务器运行：

```bash
# 使用Python内置服务器
python -m http.server 8000

# 使用Node.js http-server
npx http-server

# 使用Live Server扩展（VS Code）
# 安装Live Server扩展后右键HTML文件选择"Open with Live Server"
```

### 3. 查看和修改代码

1. **查看HTML结构**
   - 每个HTML文件都包含完整的页面结构
   - 使用浏览器开发者工具查看DOM结构

2. **查看JavaScript逻辑**
   - 对应的JS文件包含所有JavaScript代码
   - 代码包含详细注释，解释实现原理

3. **修改和实验**
   - 可以直接修改HTML或JS文件
   - 刷新浏览器查看变化
   - 尝试修改参数和逻辑，观察结果

## 🐛 常见问题

### Q: 为什么某些示例在Chrome中运行正常，但在其他浏览器中有问题？
A: 部分高级特性（如ES2020+）可能在旧版浏览器中不支持。建议使用最新版本的现代浏览器（Chrome、Firefox、Safari、Edge）。

### Q: 可以在移动设备上运行这些示例吗？
A: 大部分示例是响应式设计的，可以在移动设备上运行。但对于复杂的交互体验，桌面浏览器会更好。

### Q: 如何在这些示例中使用外部库？
A: 可以在HTML文件中添加CDN链接或通过npm安装库。修改后注意更新相关的JavaScript代码。
