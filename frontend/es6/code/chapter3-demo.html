第3章：箭头函数与函数增强 - 演示

1. 传统函数与箭头函数的对比

// 传统函数声明 function add(a, b) { return a + b; } // 箭头函数 const subtract = (a, b) => a - b;

2. 箭头函数中的this绑定

const person = { name: 'Alice', // 传统函数中的this sayNameTraditional: function() { setTimeout(function() { console.log(this.name); // undefined }, 10); }, // 箭头函数中的this sayNameArrow: function() { setTimeout(() => { console.log(this.name); // 'Alice' }, 20); } };

3. 默认参数

function greet(name = 'Guest', age = 0) { return `Hello, ${name}! You are ${age} years old.`; }

4. 剩余参数

function sum(...numbers) { return numbers.reduce((total, num) => total + num, 0); }

5. 扩展运算符

数组操作

const arr1 = [1, 2, 3]; const arr2 = [4, 5, 6]; const combined = [...arr1, ...arr2];

对象操作

const person = { name: 'Alice', age: 30 }; const employee = { ...person, position: 'Developer' };

6. 实际应用：数组方法与箭头函数

const users = [ { id: 1, name: 'Alice', age: 30 }, { id: 2, name: 'Bob', age: 25 }, { id: 3, name: 'Charlie', age: 35 } ]; // 使用箭头函数和数组方法 const userNames = users.map(user => user.name); const olderUsers = users.filter(user => user.age > 30);

7. 实际应用：解决回调中的this问题

这个示例展示了如何使用箭头函数解决回调函数中的this绑定问题。

const counter = { count: 0, start: function() { // 使用箭头函数确保this指向counter对象 setInterval(() => { this.count++; console.log(`Count: ${this.count}`); }, 1000); } };

8. 实际应用：使用对象参数处理多个可选参数

function createUser(options = {}) { const { name = 'Anonymous', age = 0, email = null, isActive = true } = options; return { name, age, email, isActive }; }
