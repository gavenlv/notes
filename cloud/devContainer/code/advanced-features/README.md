# 高级特性演示

这个示例演示DevContainer的高级特性，包括多语言环境、Docker in Docker等。

## 功能特性

- ✅ 多语言开发环境（Node.js + Python）
- ✅ Docker in Docker支持
- ✅ 特权模式运行
- ✅ 自定义构建脚本
- ✅ 高级功能特性
- ✅ 开发工具链集成

## 快速开始

1. 在VS Code中打开此目录
2. 按`Ctrl+Shift+P`，输入"Reopen in Container"
3. 等待容器构建完成
4. 运行 `./dev-tools.sh` 检查环境

## 高级特性

### 1. 多语言环境
- Node.js 18
- Python 3.11
- 支持同时开发多种语言项目

### 2. Docker in Docker
- 在容器内运行Docker
- 支持容器编排和测试
- 完整的CI/CD环境

### 3. 特权模式
- 访问宿主机Docker socket
- 支持高级容器操作
- 开发和生产环境一致性

## 学习要点

通过这个示例，你可以学习：

1. 复杂环境配置
2. 特权模式使用
3. Docker in Docker技术
4. 多语言工具链集成
5. 自定义构建脚本

## 使用场景

- 多语言微服务开发
- 容器化应用测试
- CI/CD流水线开发
- 复杂开发环境配置
