# Flink 0基础入门到专家学习指南

## 学习路径

本学习指南提供了从 Flink 零基础到专家级别的完整学习路径，按照难度递增的顺序组织内容。建议按照以下顺序学习：

1. **day01-installation** - Flink 环境搭建
2. **day02-introduction** - Flink 基础概念介绍
3. **day03-basic-operations** - Flink 基础操作
4. **day04-data-streams** - 数据流编程
5. **day05-state-management** - 状态管理
6. **day06-window-operations** - 窗口操作
7. **day07-connectors** - 数据源和连接器
8. **day08-sql-table-api** - SQL 和 Table API
9. **day09-advanced-topics** - 高级主题
10. **day10-performance-tuning** - 性能优化
11. **day11-monitoring** - 监控和运维
12. **day12-production-deployment** - 生产环境部署
13. **day13-case-studies** - 案例研究
14. **day14-troubleshooting** - 故障排查

## 学习资源

- [Apache Flink 官方文档](https://flink.apache.org/docs/stable/)
- [Flink 官方 GitHub](https://github.com/apache/flink)
- [Flink 中文社区](https://flink-china.org/)

## 学习建议

1. **理论结合实践** - 每个概念都有对应的代码示例，建议实际运行
2. **循序渐进** - 按照学习路径逐步深入，不要跳过基础内容
3. **多做练习** - 通过实际项目巩固所学知识
4. **参与社区** - 加入 Flink 社区，分享经验和提问

开始你的 Flink 学习之旅吧！
