## 🚀 如何使用本课程

### 1. 开始学习

```bash
# 进入课程目录
cd technologies/docker

# 查看主目录
cat README.md

# 查看快速开始
cat QUICK_START.md

# 从第1章开始
cd chapter01-introduction
cat README.md
```

### 2. 运行代码示例

```powershell
# Windows PowerShell
cd chapter01-introduction\code
.\run-examples.ps1
```

```bash
# Linux/Mac
cd chapter01-introduction/code
chmod +x run-examples.sh
./run-examples.sh
```

### 3. 完成练习

```bash
# 查看练习题
cd chapter01-introduction/exercises
cat exercises.md

# 完成每个练习
# 验证结果
```

```bash

```
