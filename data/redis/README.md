# Redis 从入门到专家教程

本教程将带您从零开始深入学习Redis，从基础概念到高级应用，涵盖Redis的各个方面。

## 教程结构

- **第1章**: Redis简介与安装 - 了解Redis的基本概念和安装方法
- **第2章**: Redis数据结构与命令 - 掌握Redis的核心数据结构和使用方法
- **第3章**: Redis持久化与复制 - 理解数据持久化和主从复制机制
- **第4章**: Redis高级特性与事务 - 学习高级功能和事务处理
- **第5章**: Redis集群与高可用 - 掌握集群部署和高可用方案
- **第6章**: Redis性能优化与监控 - 学习性能调优和监控方法
- **第7章**: Redis应用场景与最佳实践 - 了解实际应用场景和最佳实践
- **第8章**: Redis扩展与源码分析 - 深入Redis源码和扩展开发

## 代码示例

所有代码示例都位于 `code/` 目录中，包含：
- 基础操作示例
- 高级应用案例
- 性能测试脚本
- 监控工具

## 学习要求

- 基本了解Linux/Unix命令行操作
- 了解基本的网络概念
- 有编程经验者更佳

## 快速开始

1. 按照第1章安装Redis
2. 运行第2章的示例代码
3. 逐步学习后续章节
