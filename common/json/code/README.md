## 运行要求

### HTML文件
- 现代Web浏览器（Chrome、Firefox、Edge、Safari）
- 无需额外依赖

### JavaScript文件
- Node.js环境或浏览器环境
- 如需运行完整示例，使用Node.js: `node 4.1-json-javascript.js`

### Python文件
- Python 3.6或更高版本
- 无需额外依赖（使用标准库）

## 如何使用

1. **HTML交互式演示**：
   - 直接在浏览器中打开HTML文件
   - 按照页面提示进行交互操作
   - 查看实时结果和反馈

2. **JavaScript示例**：
   - 在Node.js环境中运行：`node 4.1-json-javascript.js`
   - 或在浏览器中通过script标签引入

3. **Python示例**：
   - 在终端中运行：`python 5.1-validation-schema.py`
   - 查看控制台输出和生成的文件

4. **YAML和JSON示例**：
   - 使用YAML/JSON编辑器查看
   - 或使用在线YAML/JSON验证器测试

## 学习路径

1. 从第1章开始，了解JSON的基本概念和结构
2. 学习第2章的语法规则，掌握JSON的正确编写方式
3. 深入第3章，理解JSON的各种数据类型和结构
4. 在第4章学习如何在JavaScript中使用JSON
5. 掌握第5章的验证和高级特性
6. 学习第6章的安全最佳实践
7. 探索第7章的JSON扩展和生态系统
8. 最后，通过第8章了解企业级应用
