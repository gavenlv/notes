## 常见问题

### YAML是什么？

YAML（YAML Ain't Markup Language）是一种人类友好的数据序列化语言，常用于配置文件、数据交换等场景。

### YAML与JSON的区别是什么？

- YAML更易读，支持注释
- JSON更严格，解析速度更快
- YAML功能更丰富，支持更复杂的数据结构

### 如何验证YAML语法？

可以使用在线YAML验证器或编程语言中的YAML解析库来验证语法。
