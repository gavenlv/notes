# 第3章：面向对象编程

## 学习目标

本章将深入讲解Scala的面向对象编程特性，包括：
- 类和对象的定义
- 构造器和访问修饰符
- 继承和多态
- 抽象类和特质（Traits）
- 对象和伴生对象
- 包和导入
- 高级类型系统特性

## 环境要求

- Scala 2.13.x 或 3.x
- sbt 1.8.x
- 完成第1章和第2章的学习

## 实践练习

### 练习1：基础类设计
设计一个简单的银行账户系统，包含：
- 基本账户类
- 储蓄账户和支票账户子类
- 账户操作方法（存款、取款、查询余额）

### 练习2：特质应用
创建一组特质来表示动物的特征：
- 可移动特质
- 可发声特质
- 食性特质
- 实现具体的动物类

### 练习3：设计模式实现
使用Scala实现以下设计模式：
- 单例配置管理器
- 简单工厂模式
- 观察者模式

## 学习建议

1. **理论结合实践**：每个概念都要通过代码示例来验证
2. **对比学习**：与Java等其他面向对象语言对比，理解Scala的独特之处
3. **深入理解特质**：特质是Scala的重要特性，需要重点掌握
4. **设计模式应用**：学习如何在Scala中实现经典设计模式
