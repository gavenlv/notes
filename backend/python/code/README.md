## 🚀 快速开始

### 运行示例代码

```bash
# 进入第1章目录
cd code/chapter01

# 运行Hello World
python hello.py

# 运行计算器
python calculator.py

# 检查环境
python env_check.py
```

### 创建虚拟环境(推荐)

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

## 💡 学习建议

1. **亲手输入**: 不要复制粘贴,自己敲一遍代码
2. **理解原理**: 运行代码前先思考会输出什么
3. **修改实验**: 尝试修改代码,观察结果变化
4. **查看文档**: 遇到不懂的函数,查看官方文档
5. **做好笔记**: 记录重点和易错点

## 🐛 遇到问题?

1. 检查Python版本是否 >= 3.8
2. 确保文件编码为UTF-8
3. 检查缩进是否正确(使用4个空格)
4. 查看完整的错误提示信息
