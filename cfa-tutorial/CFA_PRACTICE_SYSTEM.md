## 1. 分级练习题库 | Graded Practice Question Bank

### 1.1 基础练习（难度：★☆☆）

#### 数量方法基础题
**题目1：简单现值计算**
```
如果5年后能收到10000元，年利率为5%，那么现值是多少？
A) 7835.26元
B) 8626.09元  
C) 9523.81元
D) 10000元
```
**答案：A**
**解析：** PV = 10000 / (1+0.05)^5 = 10000 / 1.2763 = 7835.26元
**考试技巧：** 使用金融计算器：N=5, I/Y=5, FV=10000, CPT PV

**题目2：有效年利率计算**
```
年利率6%，每半年复利一次，有效年利率是多少？
A) 6.00%
B) 6.09%
C) 6.18%
D) 6.30%
```
**答案：B**
**解析：** EAR = (1 + 0.06/2)^2 - 1 = (1.03)^2 - 1 = 1.0609 - 1 = 6.09%

#### 财务报表基础题
**题目3：ROE计算**
```
某公司净利润500万元，股东权益2000万元，ROE是多少？
A) 20%
B) 25%
C) 30%
D) 40%
```
**答案：B**
**解析：** ROE = 净利润/股东权益 = 500/2000 = 25%

### 1.2 中级练习（难度：★★☆）

#### 数量方法中级题
**题目4：年金现值计算**
```
每年末收到1000元，持续10年，年利率8%，年金现值是多少？
A) 6710.08元
B) 7235.82元
C) 8143.25元
D) 9263.89元
```
**答案：A**
**解析：** PVA = 1000 × [(1 - (1+0.08)^-10)/0.08] = 1000 × 6.71008 = 6710.08元

**题目5：投资组合风险**
```
两个资产相关系数为0.3，权重各50%，标准差分别为15%和20%，组合标准差是多少？
A) 12.5%
B) 13.8%
C) 15.2%
D) 17.5%
```
**答案：B**
**解析：** σp = √[0.5²×0.15² + 0.5²×0.20² + 2×0.5×0.5×0.15×0.20×0.3] = 13.8%

#### 财务报表中级题
**题目6：杜邦分析**
```
某公司净利润率8%，资产周转率1.5，权益乘数2，ROE是多少？
A) 12%
B) 18%
C) 24%
D) 30%
```
**答案：C**
**解析：** ROE = 8% × 1.5 × 2 = 24%

### 1.3 高级练习（难度：★★★）

#### 数量方法高级题
**题目7：期权定价**
```
股票价格50元，执行价格55元，无风险利率5%，波动率30%，期限1年，看涨期权价值是多少？
A) 3.52元
B) 4.87元
C) 5.63元
D) 6.25元
```
**答案：B**
**解析：** 使用Black-Scholes模型计算，d1=0.15, d2=-0.15, N(d1)=0.5596, N(d2)=0.4404, C=4.87元

**题目8：债券久期**
```
5年期债券，面值1000元，票面利率6%，到期收益率7%，麦考利久期是多少？
A) 4.2年
B) 4.5年
C) 4.8年
D) 5.0年
```
**答案：A**
**解析：** 通过现金流贴现计算，麦考利久期约4.2年

## 2. 专题练习模块 | Topic-Based Practice Modules

### 2.1 职业道德专题

#### 利益冲突场景题
**题目9：个人投资冲突**
```
你正在分析某公司股票，但你个人持有该公司大量股票。你应该：
A) 继续分析，但降低推荐评级
B) 披露利益冲突，继续分析
C) 避免参与该股票的分析
D) 卖出个人持有的股票
```
**答案：C**
**解析：** 根据CFA职业道德准则，重大利益冲突时应避免参与相关分析

**题目10：客户适当性**
```
你的客户风险承受能力低，但要求投资高风险的科技股。你应该：
A) 满足客户要求
B) 解释风险，建议更合适的投资
C) 拒绝提供服务
D) 要求客户签署风险确认书
```
**答案：B**
**解析：** 投资建议必须适合客户情况，应解释风险并提供适当建议

### 2.2 财务报表专题

#### IFRS vs US GAAP差异题
**题目11：研发费用处理**
```
根据IFRS，研发支出应该：
A) 全部费用化
B) 全部资本化
C) 研究阶段费用化，开发阶段符合条件的资本化
D) 开发阶段费用化，研究阶段资本化
```
**答案：C**
**解析：** IFRS允许开发阶段符合条件的支出资本化

**题目12：存货计价**
```
根据IFRS，哪种存货计价方法被禁止？
A) FIFO
B) LIFO
C) 加权平均法
D) 个别计价法
```
**答案：B**
**解析：** IFRS禁止使用LIFO方法
