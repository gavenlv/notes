## 代码与数据示例

每个章节都配有相应的代码示例和数据文件，保存在`code`目录下，方便您跟随教程进行实际操作。

## 学习路径建议

1. 初学者：按章节顺序学习，从第1章开始
2. 有基础者：可以跳过基础章节，直接从第3章或第4章开始
3. 高级用户：重点关注第6-8章的高级功能和企业级应用

## 学习目标

完成本教程后，您将能够：
- 熟练使用Tableau进行数据可视化
- 掌握数据连接、清洗和转换技能
- 创建具有洞察力的仪表盘和故事
- 运用高级分析功能进行数据挖掘
- 在企业环境中部署和管理Tableau解决方案

## 软件版本要求

- Tableau Desktop 2023.1或更高版本
- Tableau Server 2023.1或更高版本（可选）
