# BIRT报表开发教程

本教程将帮助您从零基础开始学习BIRT（Business Intelligence and Reporting Tools），逐步成为BIRT报表开发专家。

## 教程内容

本教程包含以下章节，每章都包含详细的概念解析、实际应用代码和最佳实践：

1. [BIRT基础介绍与环境搭建](1-BIRT基础介绍与环境搭建.md) - 了解BIRT的基本概念、发展历史和应用场景，并完成开发环境搭建
2. [数据源与数据集](2-数据源与数据集.md) - 学习如何连接各种数据源，创建和管理数据集
3. [报表设计与布局](3-报表设计与布局.md) - 掌握报表设计器的使用，学习各种布局技巧和组件应用
4. [图表与可视化](4-图表与可视化.md) - 探索BIRT的图表功能，创建各种数据可视化效果
5. [参数化与交互式报表](5-参数化与交互式报表.md) - 学习如何创建参数化报表，实现用户交互功能
6. [脚本与事件处理](6-脚本与事件处理.md) - 掌握JavaScript脚本编程，实现高级报表功能
7. [部署与集成](7-部署与集成.md) - 学习BIRT报表的部署方式和与其他系统的集成方法

## 学习建议

1. **循序渐进**：建议按照章节顺序学习，每一章都建立在前一章的基础上
2. **动手实践**：每章都配有可运行的代码示例，强烈建议边学边实践
3. **深入理解**：教程中不仅讲解如何使用，还深入解释了工作原理，适合不同水平的学习者
4. **案例导向**：通过实际案例演示BIRT在各种场景下的应用

## 前置知识

- 基本的数据库知识（SQL语言）
- 了解Java和Web应用基础（可选，但有助于理解BIRT集成）
- 基本的编程概念

## 代码示例

每个章节的代码示例位于对应的`code`子目录中，包含：
- 完整的报表设计文件(.rptdesign)
- 示例数据库脚本
- Java集成代码
- Web应用示例
