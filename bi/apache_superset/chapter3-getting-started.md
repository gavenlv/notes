# 第三章：快速上手指南

## 3.1 登录系统

完成安装后，打开浏览器访问 `http://localhost:8088`，您将看到 Superset 的登录界面。

![Superset 登录界面](images/login.png)

使用在安装过程中创建的管理员账户登录：
- **用户名**：admin
- **密码**：admin（或您设置的密码）

## 3.2 界面概览

登录后，您将进入 Superset 的主界面，主要包括以下几个区域：

### 顶部导航栏
- **Home**：主页
- **Dashboards**：仪表板列表
- **Charts**：图表列表
- **Data**：数据管理
- **SQL Lab**：SQL 编辑器
- **Settings**：系统设置

### 侧边栏菜单
- **Recent**：最近访问的项目
- **Favorites**：收藏的项目
- **Examples**：示例内容

### 主工作区
显示当前页面的主要内容，如仪表板、图表列表等。

## 3.3 创建第一个数据源

### 步骤 1：进入数据源管理

点击顶部导航栏的 **Data** → **Database Connections**，进入数据库连接管理页面。

### 步骤 2：添加数据库连接

点击右上角的 **+ Database** 按钮，填写数据库连接信息：

1. **Database Name**：给数据库起个名字，例如 "示例数据库"
2. **SQLAlchemy URI**：数据库连接字符串，例如：
   - SQLite: `sqlite:///path/to/database.db`
   - PostgreSQL: `postgresql://username:password@host:port/database`
   - MySQL: `mysql://username:password@host:port/database`

3. **Expose in SQL Lab**：勾选此项以允许在 SQL Lab 中使用此数据库
4. **Allow CREATE TABLE AS**：根据需要选择是否允许创建表
5. **Allow DML**：根据需要选择是否允许数据操作

点击 **Connect** 完成数据库连接。

### 步骤 3：创建数据表

数据库连接成功后，可以创建数据表：

1. 点击刚创建的数据库名称
2. 选择相应的 schema（如果适用）
3. 点击 **+ sign** 图标创建新的数据表
4. 输入表名和 SQL 查询语句
5. 点击 **Save** 保存

## 3.4 创建第一个图表

### 步骤 1：选择数据集

1. 点击顶部导航栏的 **Charts**
2. 点击右上角的 **+ Chart** 按钮
3. 选择之前创建的数据集

### 步骤 2：选择图表类型

Superset 提供了多种图表类型，包括：

- **Table View**：表格视图
- **Bar Chart**：柱状图
- **Line Chart**：折线图
- **Pie Chart**：饼图
- **Area Chart**：面积图
- **Bubble Chart**：气泡图

选择一个适合您数据的图表类型。

### 步骤 3：配置图表

在图表编辑界面中，您需要配置以下几个关键要素：

1. **Dimensions（维度）**：用于分组的字段，通常显示在 X 轴
2. **Metrics（指标）**：用于计算的数值字段，通常显示在 Y 轴
3. **Filters（过滤器）**：用于筛选数据的条件
4. **Options（选项）**：图表的样式和行为设置

### 步骤 4：预览和保存

配置完成后，点击右上角的 **Run** 按钮预览图表效果。如果满意，点击 **Save** 保存图表。

## 3.5 创建仪表板

### 步骤 1：新建仪表板

1. 点击顶部导航栏的 **Dashboards**
2. 点击右上角的 **+ Dashboard** 按钮
3. 输入仪表板名称和描述
4. 点击 **Save** 保存

### 步骤 2：添加图表

1. 在仪表板编辑界面中，点击 **+ Add Chart**
2. 选择之前创建的图表
3. 调整图表在仪表板中的位置和大小

### 步骤 3：配置交互

1. **Filter Box**：添加过滤器框，用于动态筛选数据
2. **Cross-filter**：设置图表间的联动过滤
3. **Refresh Interval**：设置自动刷新间隔

### 步骤 4：发布仪表板

配置完成后，点击 **Publish** 按钮发布仪表板，使其对其他用户可见。

## 3.6 使用 SQL Lab

### 步骤 1：进入 SQL Lab

点击顶部导航栏的 **SQL Lab** → **SQL Editor**。

### 步骤 2：编写 SQL 查询

1. 选择数据库和相应的 schema
2. 在编辑器中编写 SQL 查询语句
3. 点击 **Run** 执行查询

### 步骤 3：查看结果

查询结果会显示在下方的结果面板中，您可以：

- 查看数据表格
- 创建可视化图表
- 下载查询结果
- 保存查询为数据集

## 3.7 用户管理

### 创建新用户

1. 点击右上角的用户头像
2. 选择 **Manage** → **Users**
3. 点击 **+ User** 按钮
4. 填写用户信息并分配角色
5. 点击 **Save** 保存

### 角色管理

Superset 提供了以下几种默认角色：

- **Admin**：系统管理员，拥有全部权限
- **Alpha**：高级用户，可以访问大部分功能
- **Gamma**：普通用户，只能查看和创建自己的内容
- **Public**：公共角色，未登录用户的角色
- **granter**：权限授予者

通过本章的学习，您已经掌握了 Superset 的基本使用方法，包括：

1. 系统登录和界面导航
2. 数据源连接和数据集创建
3. 图表创建和配置
4. 仪表板设计和发布
5. SQL Lab 的使用
6. 用户和角色管理
